##Install Homebrew
```
ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go/install)"
```

##Install Fish Shell
```
brew install fish
```

##Allow Fish as a type of shell
```
echo "/usr/local/bin/fish" | sudo tee -a /etc/shells
```

##Run configure
```
bin/configure
```
